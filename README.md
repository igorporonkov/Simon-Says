🎮 Simon Says – Игра на память на 7 языках
Классическая игра «Саймон говорит» в терминале!
Запоминай и повторяй последовательность цветов (или букв) с каждым раундом.
Проверь свою память, скорость реакции и внимание.
Реализована на семи языках программирования с единым интерфейсом, цветным выводом, звуковыми эффектами и таблицей рекордов.

🚀 Возможности
Геймплей – компьютер показывает последовательность цветов (или символов), игрок должен повторить её в правильном порядке.

Сложность – с каждым раундом последовательность увеличивается на один элемент.

Цветной вывод – каждый цвет/символ отображается соответствующим цветом в терминале (зелёный, красный, синий, жёлтый).

Звуковые эффекты – в некоторых реализациях воспроизводятся звуки для каждого цвета (через системный beep или библиотеку).

Подсказки – игрок видит, какой элемент сейчас активен.

Счёт – текущий раунд и рекорд (максимальное количество запомненных шагов).

Сохранение рекорда – лучший результат сохраняется в локальный файл.

Таймер – ограничение времени на ввод (в некоторых версиях).

Интерактивность – ввод осуществляется через клавиши (1-4 или A-D).

Выход – можно выйти в любой момент, нажав q или Ctrl+C.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Опции
Опция	Описание
-s, --speed <ms>	Скорость показа последовательности (задержка между шагами в мс, по умолчанию 800)
-r, --rounds <N>	Количество раундов для игры (0 – бесконечно, по умолчанию 0)
-t, --timeout <сек>	Таймаут на ввод (секунды, 0 – без ограничения)
-h, --help	Справка
Примеры
bash
# Запустить игру
python simon.py

# Установить скорость 500 мс и 10 раундов
python simon.py -s 500 -r 10

# Установить таймаут 3 секунды
python simon.py -t 3
Управление
Нажмите клавиши 1, 2, 3, 4 (или A, B, C, D), чтобы выбрать соответствующий цвет.

q или Ctrl+C – выход.

После ошибки игра покажет правильную последовательность и предложит начать заново.

🧠 Логика игры
Генерация последовательности – компьютер случайным образом выбирает один из 4 цветов и добавляет его к текущей последовательности.

Демонстрация – последовательность показывается по одному элементу с задержкой (цвет подсвечивается).

Ввод игрока – игрок вводит символы, соответствующие цветам, в том же порядке.

Проверка – каждый введённый символ сравнивается с соответствующим элементом последовательности.

Ошибка – если игрок ошибся, игра заканчивается, выводится правильная последовательность, обновляется рекорд.

Победа – если игрок правильно повторил всю последовательность, начинается следующий раунд с добавлением нового элемента.

✨ Дополнительные фичи
Красивый интерфейс – цветная индикация, анимация, ASCII-рамки.

Звуковое сопровождение – для каждого цвета свой звук (реализовано через beep или console.beep в некоторых языках).

Рекорд – сохраняется в файл и отображается при старте и окончании игры.

Поддержка клавиш WASD – альтернативное управление.

Режим испытания – можно ограничить количество раундов.
